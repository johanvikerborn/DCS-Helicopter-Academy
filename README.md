# DCS Helicopter Academy

A free and open-source helicopter training academy for DCS World.

The purpose of this project is to provide a structured learning path for helicopter pilots in DCS World, starting with the UH-1H Huey and expanding to other aircraft over time.

The training is designed around progressive lessons, practical missions, and real-world helicopter flying concepts.

---

## Goals

- Learn helicopter fundamentals
- Develop stable hover control
- Master low-level navigation
- Perform precision landings
- Fly realistic training exercises
- Build confidence before entering combat scenarios

---

## Training Roadmap

### Phase 1 - Fundamentals

- Lesson 001 - First Lift
- Lesson 002 - Hover Control
- Lesson 003 - Air Taxi

### Phase 2 - Precision Flying

- Lesson 004 - River Run
- Lesson 005 - Bridge Run
- Lesson 006 - Precision Landing

### Phase 3 - Advanced Operations

- Lesson 007 - Mountain Flying
- Lesson 008 - Rescue Operations
- Lesson 009 - Night Flying
- Lesson 010 - NOE Flying

## Mission Philosophy

The goal is not simply to complete missions.

The goal is to become a skilled helicopter pilot.

Every lesson focuses on one specific skill and builds upon previous lessons.
